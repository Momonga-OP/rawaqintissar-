🔧 CUSTOMIZATION GUIDE
═══════════════════════════════════════════════════════════════════

To Change Colors:
  Edit css/style.css, modify the :root variables at the top

To Add New Products:
  1. Add product image to images/ folder
  2. Copy a product card in products.html
  3. Update image src, category, and data-translate keys
  4. Add translations in js/translations.js

To Modify Translations:
  Edit js/translations.js and update the translations object

To Change Fonts:
  Replace Google Fonts links in HTML head sections

To Add New Pages:
  1. Create new HTML file
  2. Copy header/footer from existing pages
  3. Add navigation link
  4. Update translations if needed
