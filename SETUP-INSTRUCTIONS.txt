## ğŸš€ Getting Started

### Setup Instructions

1. **Add Your Images**
   - Add product images to the `images/` folder
   - Required images:
     - `womens-collection.jpg`, `mens-collection.jpg`, `kids-collection.jpg`, `sets-collection.jpg` (Featured collections)
     - `product-1.jpg` through `product-9.jpg` (Individual products)
     - `about-story.jpg`, `about-mission.jpg`, `about-image.jpg` (About page)
     - `gallery-1.jpg` through `gallery-12.jpg` (Gallery page)

2. **Update Contact Information**
   - Replace `+212 XXX XXX XXX` with your actual phone number in all HTML files
   - Update WhatsApp link: Replace `212XXXXXXXXX` with your WhatsApp number
   - Update social media links:
     - Instagram: `https://instagram.com/rawaqintissar`
     - TikTok: `https://tiktok.com/@rawaqintissar`
   - Update email: `info@rawaqintissar.com`

3. **Google Maps Integration**
   - In `contact.html`, replace the Google Maps iframe embed URL with your actual location

4. **Launch the Website**
   - Simply open `index.html` in a web browser
   - Or host on a web server for production

### Adding New Products

Edit `products.html` and add new product cards following this structure:

```html
<div class="product-card" data-category="womens">
    <div class="product-image">
        <img src="images/your-image.jpg" alt="Product Name">
        <div class="product-overlay">
            <button class="btn-icon zoom-btn" data-image="images/your-image.jpg">
                <i class="fas fa-search-plus"></i>
            </button>
            <a href="#whatsapp" class="btn-icon whatsapp-btn">
                <i class="fab fa-whatsapp"></i>
            </a>
        </div>
    </div>
    <div class="product-info">
        <h3 class="product-title" data-translate="product_name_key">Product Name</h3>
        <p class="product-desc" data-translate="product_desc_key">Description</p>
    </div>
</div>
```

Then add translations to `js/translations.js`.
